Getting Started

üì• Installation

How to install Extended History:

  1. Visit the Chrome Web Store
  2. Click "Add to Chrome"
  3. Confirm the installation
  4. The extension will automatically replace your default history page

üöÄ First Steps

After installation:

  1. Press Ctrl+H (Windows/Linux) or Cmd+H (Mac) to open your history
  2. Click the extension icon for quick access to recent history
  3. Customize your settings by clicking the Settings button
  4. Optionally backfill existing Chrome history into the extension

Recommended settings:

  • Set your preferred retention period (default: 3650 days)
  • Choose your theme (dark or light mode)
  • Customize accent colors to match your style
  • Enable auto-save sessions if desired

Common Issues & Solutions

üîÑ History Not Tracking

Problem: New pages aren't being added to history

Solutions:

  • Check that the extension is enabled in Chrome Extensions
  • Ensure you're not in Incognito mode (extensions don't track incognito by default)
  • Try disabling and re-enabling the extension
  • Verify the site URL is not a Chrome internal page (chrome://...)

‚è±Ô∏è Time Tracking Issues

Problem: Time tracking seems inaccurate

Solutions:

  • Time is only tracked when the window is focused and tab is active
  • Background tabs are not tracked
  • Minimized windows don't count toward time tracking
  • Try clicking "Flush Time" in settings to commit current session

üíæ Storage Issues

Problem: Running out of storage space

Solutions:

  • Reduce retention period in settings
  • Delete old history entries you no longer need
  • Export important data before clearing
  • The extension requests unlimited storage but Chrome may still limit it

üîç Search Not Working

Problem: Can't find specific history entries

Solutions:

  • Check your date range filter
  • Try searching for domain instead of full URL
  • Use the "Any time" filter to search all history
  • Verify the entry wasn't deleted

üìë Sessions Not Saving

Problem: Auto-save or manual session save fails

Solutions:

  • Check browser download permissions
  • Ensure you have tabs open to save
  • Try manually exporting a session first
  • Check if auto-save interval is properly configured

üé® Theme or UI Issues

Problem: Interface looks broken or colors are wrong

Solutions:

  • Try switching between dark and light themes
  • Reset accent colors to default
  • Clear browser cache and reload
  • Check for browser updates

Feature Guides

üîç Filtering and Search

How to filter your history effectively:

  • Date Range: Use the date picker to select specific time periods
  • Search: Type in the search box to filter by URL or title
  • Domain Filter: Click on a domain to see only entries from that site
  • Quick Filters: Use "Today", "Yesterday", "Last 7 Days" for common ranges
  • Combine Filters: Use date range + search together for precise results

‚è±Ô∏è Time Tracking

Understanding time tracking:

  • Active Time Only: Only counts time when tab is active and window is focused
  • Per Domain: Time is aggregated by domain (e.g., all youtube.com time together)
  • Daily Breakdown: View time spent per day in the Activity panel
  • Statistics: See top sites by time spent in the last 7, 30, or 90 days
  • Privacy: All time data is stored locally, never shared

üìë Session Management

Working with sessions:

  • Current Session: View all currently open tabs
  • Save Session: Manually save current tabs as a named session
  • Auto-Save: Configure automatic periodic session saves
  • Restore Session: Open all tabs from a previous session
  • Export Session: Download session as HTML file for offline access
  • Max Sessions: Set how many past sessions to keep (1-20)

üíæ Export and Import

Data portability:

  • Export History: Download all history as JSON file
  • Export Bookmarks: Save bookmarks in JSON or HTML format
  • Import Data: Upload previously exported JSON to restore history
  • Backup Strategy: Export regularly to keep backups of your data
  • Migration: Use export/import to move data between computers

üóëÔ∏è Deletion Options

How to delete history:

  • Single Entry: Click the trash icon on any history item
  • Selection: Select multiple items with checkboxes and delete batch
  • By Filter: Filter history, then delete all matching entries
  • Date Range: Delete all history within a specific date range
  • Clear All: Nuclear option - removes all history at once
  • Cookies & Cache: Optionally clear cookies/cache alongside history

Data and Privacy

üîí Where is my data stored?

All your data is stored locally on your device using Chrome's local storage API. Nothing is sent to external servers or the cloud.

üö´ What data is collected?

Only browsing history (URLs, titles, timestamps), time tracking, and session data. No personal information, passwords, or form data is collected.

üîê How secure is my data?

Your data is as secure as your device. We recommend using device encryption and keeping your operating system updated.

‚ôªÔ∏è How do I delete all my data?

Use "Clear All" in settings, or simply uninstall the extension. Uninstalling permanently removes all data stored by Extended History.
